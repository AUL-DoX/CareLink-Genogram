# CareLink Genogram

介護のフェイスシート等で使う **ジェノグラム（ジェノマップ）を爆速で作成**するためのシンプルWebツールです。  
介護ソフトの融通の利かないジェノグラム機能、または Excel 図形描画（○□線の貼り付け・微調整）による手間を減らすことを目的にしています。

- 入力：**名前 / 続柄 / 関係機関**
- 出力：**ジェノグラム画像**
- 使い方：**画像を Excel / Googleスプレッドシートにコピペ**するだけ

> 本ツールは「保存・管理」よりも「とにかく早く作る」を優先しています。

---

## Demo / Usage
1. ブラウザでアプリを開く
2. 名前・続柄・関係機関を入力
3. 生成されたジェノグラム画像を **Excel / スプレッドシートへ貼り付け**

---

## Install / Run（ローカルで使う）
このプロジェクトは静的HTMLとして動作します。

### 方法A：そのまま開く
`carelink-genogram.html` をダブルクリックしてブラウザで開きます。

### 方法B：簡易サーバ（推奨）
一部ブラウザ制限を避けるため、ローカルサーバ起動を推奨します。
